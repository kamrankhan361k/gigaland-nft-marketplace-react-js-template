How to Install
  1. Node.js and NPM : You can download Node.js from https://nodejs.org. please use Recommended For Most Users 16.13.2 LTS NPM comes bundled with Node.js.
  2. NPM : You must have to npm install packages. You can do this by running npm install from the root of your project to install all the necessary dependencies.

    Open a terminal(Windows Command Prompt or PowerShell).

  3. Run npm start for a dev server. Navigate to http://localhost:3000/ . The app will automatically reload if you change any of the source files.
  4. Run npm run build to build the project. The build artifacts will be stored in the build/ directory. Use the npm run build --prod flag for a production build.
If not work using NPM try using Yarn
  1. add SKIP_PREFLIGHT_CHECK=true in file .env
  2. Remove node module command rm -rf node_modules
  3. Delete package-lock.json
  4. Install Yarn ( https://yarnpkg.com/en/docs/install ).
  5. Run yarn install
  6. Run yarn add webpack@4.44.2
  7. Run yarn run start from the folder

Custom background image for div/section

To make custom background image without touching css file, you can use code style="url(your-image-url)" within your div/section tag. See sample below:

style={{backgroundImage: `url(${'./img/background/4.jpg'})`}}

<section id='profile_banner' className='jumbotron breadcumb no-bg' style={{backgroundImage: `url(${'./img/background/4.jpg'})`}}>
    <div className='mainbreadcumb'>
    </div>
  </section>

Create Columns

For a simple two column layout, create a .row and add the appropriate number of .col-md-* columns. As this is a 12-column grid, each .col-md-* spans a number of those 12 columns, and should always add up to 12 for each row (or the number of columns in the parent).

Please read more for Bootstrap grid system here: https://getbootstrap.com/docs/4.5/layout/grid/
